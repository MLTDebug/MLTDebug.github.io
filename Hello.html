Checkout some git commands here!

  • ls-see the files in the repository
  • git add -adds a file in the repository folder to the repository

  • git pull- sync all the online github.com changes to your local files.
  • git commit -m “message” -save your changes to the local repository.
  • git reset --hard -goes back to the previous versions
  • git reset --hard origin/master-
  • git branch-see the current branch
  • git checkout -b branch-create and switch to a new branch
  • git checkout branch-switch to an existing branch
  • git merge branch-merge the current branch with another branch
  • git log-shows a list of history versions
